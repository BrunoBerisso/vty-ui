vty-ui is a high-level user interface library for applications running in terminal emulators. It provides similar functionality to what you might expect from graphical toolkits like GTK and QT.

The library provides comprehensive Haddock documentation in addition to a collection of relatively comprehensive demonstration programs; see the *-demo programs in vty-ui.cabal and their respective sources in src.
